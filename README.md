# 프로젝트 개요
   바쁜 일상으로 인해 만남의 기회가 적거나 없는 현대인들에게 사주팔자의 일주를 통한 맞춤형 매칭 서비스를 제공하여 
  시간을 절약하면서 자신의 성향과 맞는 사람들을 매칭 시켜 주어 좀 더 성공률이 높은 연애를 할수 있도록 제공.
  
# 개발환경
* SPRING BOOT 2.5.7 (Spring Security의 버전 호환 문제로 인해 프로젝트 중간에 다운그레이드 하였음.)
* JAVA 11
* JPA (Hibernate)
* MYSQL 8
* LOMBOK
* GRADLE
* SPRING SECURITY
* AWS EC2 (LINUX 2)
